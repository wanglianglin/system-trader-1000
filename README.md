# system-trader-1000
Demo project to execute system trade by using python.

## Requirement
+ pandas
+ pandas_datareader
+ matplotlib
+ click

## Running
```
python3 systemtrader/systemtrader.py <command>
```
+ *N225*     Retrieve and analyze NIKKEI225 trade history
+ *DEXJPUS*  Retrieve and analyze DEXJPUS trade history
